# Magnetic Snap Core

フロントエンドの「極上の手触り（磁気スナップ）」と、バックエンドにおける「厳密な空間バリデーション」を完全同期させる幾何学アルゴリズム・コア。

自由座標（浮動小数点数）を特定の格子座標（整数マトリクス）へ射影する際、2点間のユークリッド距離が一定の閾値（Threshold）未満であれば強制的に吸着（Snap）させる空間アライメント制御を実装しています。

## 🪐 空間アライメントの基本幾何

スナップの判定には、以下の2次元ユークリッド距離の数式をベースに、各言語のランタイム特性に最適化した処理を行っています。

$$ d = \sqrt{(x_{\text{raw}} - x_{\text{target}})^2 + (y_{\text{raw}} - y_{\text{target}})^2} $$

$d < \text{Tolerance}$ （許容値）のとき、アライメント（吸着）を確定します。

## 📂 収録コア・モジュール

### 1. iOS / Swift (`/ios-swift`)
`UIPanGestureRecognizer` と連動し、指の動き（自由座標）を追従させつつ、最寄りのグリッドインデックスを逆算。吸着圏内に入った瞬間にプレビュー用のViewを「ピタッ」と固定する、ネイティブアプリ特有の吸いつきインタラクションを制御。

### 2. Backend / Go (`/backend-go`)
クライアントからリアルタイムに送信される生のドロップ座標を受け止め、サーバーサイド側で同一の幾何学丸め（Round）および境界チェックを実施。チートや改ざんを無効化する、堅牢な空間防衛ロジック。

### 3. Web / JavaScript (`/web-js`)
ブラウザの Canvas / DOM マウスイベントに追従し、描画レートを損なうことなく「滑らかな操作感」と「正確なアライメント」を分離制御する軽量エンジン。
